## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
   bash
   git clone https://github.com/kendalgates/event-management-app.git
   cd event-management-app
   bash
   npm install
   or
   yarn install
   bash
   npm run dev
   or
   yarn dev

The application will be available at `http://localhost:5173`
